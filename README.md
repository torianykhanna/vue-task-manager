# Vue Task Manager App

A modern and responsive Todo application built with Vue 3, TypeScript, and Vite.

## Live Demo

🔗 https://torianykhanna.github.io/vue-task-manager/

---

## Technologies

- Vue 3
- TypeScript
- Vite
- SCSS
- Composition API

---

## Features

- Add new tasks
- Edit task titles
- Delete tasks
- Toggle task completion
- Toggle all tasks
- Clear completed tasks
- Filter tasks:
  - All
  - Active
  - Completed
- Error handling
- Loading states
